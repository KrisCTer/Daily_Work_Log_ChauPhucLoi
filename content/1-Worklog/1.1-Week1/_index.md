### Week 1 Achievements:
* Understood what cloud computing is, why to use AWS, and grasped the 5 main AWS service groups.
* Summarized modules, joined the FCJ community/group, and started earning credits.
* Understood the global infrastructure (Region/AZ/Edge Locations) and know how to use Console, CLI, SDK.
* AWS account is active, Budget Alert is enabled ($100 and $10 thresholds), understood different types of Budgets and cost optimization basics.
* AWS account secured (Enabled MFA for root, created Admin user for login).
* Successfully ran basic CLI commands (e.g., `aws s3 ls`).
* Installed draw.io and can draw simple AWS architecture diagrams.
* Successfully created and ran an EC2 instance with Nginx, accessed the web page via public IP, and SSH'd into the instance.
* Practiced Volume management (attached EBS, formatted & mounted to `/data`), successfully created and restored Snapshots. Understood the difference between Security Group (SG) and NACL.
* Understood AWS Support packages and know how to open a support ticket.
* Cleaned up EC2 and EBS resources to save costs; completed the Worklog and planned for Week 2.
