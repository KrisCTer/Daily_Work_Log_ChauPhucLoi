### Week 1 Achievements:
* Understood what cloud computing is, why to use AWS, and grasped the 5 main AWS service groups.
* Summarized modules, joined the FCJ community/group, and started earning credits.
* Understood the global infrastructure (Region/AZ/Edge Locations) and know how to use Console, CLI, SDK.
* AWS account is active, Budget Alert enabled ($10 threshold), understood the 4 budget types and cost optimization basics.
* AWS account secured (enabled MFA for root, created Admin user for login, not using root for daily work).
* Successfully ran basic CLI commands (e.g., `aws s3 ls`).
* Installed draw.io and can draw simple AWS architecture diagrams.
* Completed Lab 000001 (AWS Free Tier 2025) – earned initial AWS credit.
* Familiar with the AWS Billing Dashboard and Cost Explorer; completed Lab 000007 (Cost Management with AWS Budget).
* Understood AWS Support packages and know how to open a support ticket; completed Lab 000009.
* Completed the Week 1 Worklog and planned for Week 2.
