**On this page**, you will need to introduce your worklog. **How** did you complete it? How many weeks did you take to complete the program? **What** did you do in those weeks?

Typically, and as a standard, a worklog is carried out over about 3 months (throughout the internship period) with weekly contents as follows:

**Week 1:** [Getting familiar with AWS, creating a Free Tier account, and setting up basic security (MFA) and cost alerts (AWS Budgets)](1.1-week1/)

**Week 2:** [Mastering AWS networking fundamentals with Virtual Private Cloud (VPC): CIDR, Subnets, IGW/NAT, Security Groups vs NACL, VPC Peering, and Transit Gateway](1.2-week2/)

**Week 3:** [Mastering EC2 fundamentals (instance types, AMI, EBS, user data, Auto Scaling), CloudWatch monitoring, and Amazon Lightsail](1.3-week3/)

**Week 4:** [Mastering AWS storage services (S3, CloudFront, AWS Backup, VM Import/Export, Storage Gateway, FSx for Windows)](1.4-week4/)

**Week 5:** [Mastering IAM & Security in depth (Users/Groups/Roles/Policies, Cognito, KMS, Security Hub, Resource Tags, Permission Boundary)](1.5-week5/)

**Week 6:** [Deep diving into Database knowledge (RDS, Aurora, Database Migration Service) and completing PROJECT 1 – Personal Blog Website](1.6-week6/)

**Week 7:** [Building a Data Lake and mastering Data & Analytics (DynamoDB, Glue, Athena, QuickSight) — completing all 7 core Modules](1.7-week7/)

**Week 8:** [Kicking off the Genzite Capstone Project: containerizing the ai-service on ECS Fargate and fixing 3 priority bugs](1.8-week8/)

**Week 9:** [Building the Genzite Backend (NestJS) and Storage (S3) layers, with a complete end-to-end integration flow](1.9-week9/)

**Week 10:** [Finalizing the Genzite API/Frontend and integrating all system layers (Networking, Security)](1.10-week10/)

**Week 11:** [Finalizing Genzite: advanced Monitoring, basic security review (KMS, Security Hub), cost optimization, and Workshop Documentation](1.11-week11/)

**Week 12:** [Completing the Genzite Project 100% and submitting the full FCJ graduation report](1.12-week12/)
