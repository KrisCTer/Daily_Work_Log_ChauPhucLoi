**Trong trang này** bạn sẽ cần giới thiệu worklog của bạn **như thế nào**? Bạn hoàn thành chương trình trong vòng **bao nhiêu tuần**? Bạn **đã làm gì** trong các tuần đó?


Thông thường và cũng là tiêu chuẩn, một worklog được thực hiện trong khoảng 3 tháng (trong suốt thời gian thực tập) với nội dung các tuần như sau:

**Tuần 1:** [Làm quen với AWS, tạo tài khoản Free Tier, thiết lập bảo mật cơ bản (MFA) và cấu hình cảnh báo chi phí (AWS Budgets)](1.1-week1/)

**Tuần 2:** [Nắm vững nền tảng Networking trên AWS với Virtual Private Cloud (VPC): CIDR, Subnet, IGW/NAT, Security Group vs NACL, VPC Peering và Transit Gateway](1.2-week2/)

**Tuần 3:** [Nắm vững kiến thức nền tảng về EC2 (instance types, AMI, EBS, user data, Auto Scaling), giám sát CloudWatch và Amazon Lightsail](1.3-week3/)

**Tuần 4:** [Nắm vững các dịch vụ lưu trữ trên AWS (S3, CloudFront, AWS Backup, VM Import/Export, Storage Gateway, FSx for Windows)](1.4-week4/)

**Tuần 5:** [Nắm vững IAM & Security chuyên sâu (Users/Groups/Roles/Policies, Cognito, KMS, Security Hub, Resource Tags, Permission Boundary)](1.5-week5/)

**Tuần 6:** [Đào sâu kiến thức Cơ sở dữ liệu (RDS, Aurora, Database Migration Service) và hoàn thành PROJECT 1 – Personal Blog Website](1.6-week6/)

**Tuần 7:** [Xây dựng Data Lake và nắm vững Data & Analytics (DynamoDB, Glue, Athena, QuickSight) – hoàn tất toàn bộ 7 Module nền tảng](1.7-week7/)

**Tuần 8:** [Khởi động Capstone Project Genzite: containerize ai-service lên ECS Fargate và fix 3 bug ưu tiên](1.8-week8/)

**Tuần 9:** [Xây dựng Backend (NestJS) và Storage layer (S3) cho Genzite, hoàn thiện luồng tích hợp end-to-end](1.9-week9/)

**Tuần 10:** [Hoàn thiện API/Frontend và tích hợp toàn bộ các layer của Genzite (Networking, Security)](1.10-week10/)

**Tuần 11:** [Hoàn thiện Genzite: Monitoring nâng cao, rà soát bảo mật cơ bản (KMS, Security Hub), tối ưu chi phí và Workshop Documentation](1.11-week11/)

**Tuần 12:** [Hoàn thiện 100% Genzite Project và nộp báo cáo tốt nghiệp FCJ đầy đủ](1.12-week12/)
